# 🚀 PASO 1 - CREAR REPOSITORIO GITHUB

## ACCIÓN INMEDIATA:

### 1. Abrir navegador en esta URL:
👉 **https://github.com/new**

### 2. Rellenar formulario:
- **Repository name:** `mariano-maresca-exposicion`
- **Description:** `Web exposición Mariano Maresca - Memoria Viva de la Cultura Granadina`
- **Privacy:** Seleccionar "Public" (o "Private" si prefieres)
- **NO** marcar "Add a README file"
- **NO** marcar "Add .gitignore"
- **NO** marcar "Choose a license"

### 3. Click en botón verde:
👉 **"Create repository"**

### 4. COPIAR la URL que te muestra GitHub:
Ejemplo: `https://github.com/TU_USUARIO/mariano-maresca-exposicion.git`
